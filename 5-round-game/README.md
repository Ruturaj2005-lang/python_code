Overview
This is a simple Python game where the user competes against the computer by guessing numbers between 1 and 10. The game lasts for five rounds, and scores are tracked for both sides. At the end of the five rounds, the program announces the final winner.

Features
User vs Computer number guessing

5 rounds of play

Score tracking for both sides

Final winner announcement

How It Works
The computer randomly selects a secret number between 1 and 10.

The user enters their guess.

If the guess matches the secret number, the user wins the round.

Otherwise, the computer wins the round, and the correct number is revealed.

Scores are updated after each round.

After five rounds, the program displays the final scores and declares the winner.
